## Place an order 
When placing orders you create a post to endpoint: 

``` 
POST https://gateway.saxobank.com/sim/openapi/trade/v2/orders

{
	"Uic": 16,
	"BuySell": "Buy",
	"AssetType": "FxSpot",
	"Amount": 100000,
	"OrderPrice": 7,
	"OrderType": "Limit",
	"OrderRelation": "StandAlone",
	"ManualOrder": true,
	"OrderDuration": {
		"DurationType": "GoodTillCancel"
	}
}
```
